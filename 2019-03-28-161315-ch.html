ubuntu crontab日志存储

ubuntu的crontab的日志默认存储在/var/log/syslog中,如要查看的话,可以使用grep CRON /var/log/syslog查看

也可配置,输出到其他路径下.

# 打开文件
/etc/rsyslog.d/50-default.conf
# 查找cron.* 并且注释掉
# 执行服务
sudo service rsyslog restart
# 这样就可以看见日志输出到cron.log中了
/var/log/cron.log
